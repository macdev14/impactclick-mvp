How It Works

When users click the widget button:

  1. reCAPTCHA verification is performed
  2. A donation is triggered to the specified NGO
  3. The user receives immediate feedback
  4. The interaction is tracked for analytics

Note: This template uses demo/test configuration. For production use, replace the demo values with your actual campaign and NGO IDs, and use a production reCAPTCHA key.
